- **裁判自偏**:裁判与被评为同款模型(gpt-5.4-mini),可能偏爱同源文风;更严谨应换更强模型当裁判或双裁判(见 stage-6-design.md §9.2)。

- **单跑随机性**:未固定温度多次取均值;生产评估应多跑取中位数。
